`Awaited` 문법을 통해서 Promise 타입을 벗겨낼 수 있다.

```ts 
// A = string
type A = Awaited<Promise<string>>;

// B = number
type B = Awaited<Promise<Promise<number>>>;
```

참고 자료

- https://devblogs.microsoft.com/typescript/announcing-typescript-4-5/#awaited-type
- https://stackoverflow.com/questions/48944552/typescript-how-to-unwrap-remove-promise-from-a-type

```js
const generateTxtFile = (data) => {
  const blob = new Blob([JSON.stringify(data)], {
    type: 'text/plain;charset=UTF-8',
  });
  const textFile = window.URL.createObjectURL(blob);
  return textFile;
};
```

- `charset=UTF-8`을 붙여야 한글이 깨지지 않는다.

[File](https://developer.mozilla.org/ko/docs/Web/API/File)을 활용하는 방법

```js
const generateTxtFile = (data) => {
  const fileName = `test.txt`;
  const txtFile = new File([JSON.stringify(data)], fileName, {
    type: 'text/plain;charset=UTF-8',
  });
  return txtFile;
};
```

```js
window.open(generateTxtFile('한글'), '_blank');
```
